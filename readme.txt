1.0 Installation
    > Install Chrome Live Reload extension
    > Enable the Live Reload extension by clicking on it in top right corner of browser
    > npm install

2.0 Development
    > npm run dev

2.1 Changing API Endpoint
    > set API_URL=http://localhost:2001
    > npm run dev

    Note: You will see in your command window what api url is set (Loading with http://localhost:2001 as api endpoint)

3.0 Running Angular 8
    > npm run dev-new
